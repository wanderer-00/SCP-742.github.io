Для описания команд будут использоваться некоторые символы.

[ ] – в этих скобках находится необязательный параметр какой либо команды
| – этим символом разделяются альтернативные параметры команды(или то или то)
В дальнейшем заменить на синтаксис!!!

show ip eigrp neighbors [x] [details] - можно посмотреть таблицу соседей протокола EIGRP. X это номер автономной системы, слово [details] расширяет вывод
show ip eigrp interfaces [x] [details] – отображает информацию протокола EIGRP для каждого интерфейса.Номер автономной системы ограничивает вывод, слово details расширяет вывод
show ip eigrp topology [x] [ip-address][mask]] [all-links] – выводит все маршруты, резервные и основные.С помощью необязательных параметров можно ограничить(отфильтровать) вывод на основе автономной системы, или адреса сети.Необязательный параметр [all-links] отображает не только активные маршруты, но и все имеющиеся маршруты топологии сети протокола EIGRP.
show ip eigrp traffic [x] – показывает число отправленных/полученных пакетов протокола EIGRP. Можно отфильтровать вывод задав [x](номер автономной системы).

Команды для отладки протокола EIGRP.

debug eigrp fsm – этой командой можно посмотреть как работает алгоритм DUAL,как срабатывает резервный маршрут, как добавляются и удаляются маршруты.
debug eigrp packets – показывает передачу и прием пакетов протокола EIGRP.Например пакеты приветствия, обновления маршрутов, запросы или ответы на запрос.В пакетах используются номера последовательности и номера подтверждений используемые фирменным алгоритмом надежной доставки данных протокола EIGRP.
